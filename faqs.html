Frequently Asked Questions

What is this website for?

This website helps users detect deepfake media by analyzing images and videos for signs of manipulation.

How do I reset my password?

If you forget your password, you can reset it by clicking the "Forgot Password" link on the login page and answering your security question.

What file formats are supported?

We currently support PNG, JPEG, JPG for images, and MP4, AVI for videos.

Is my data secure?

Yes, we prioritize your data's security by following best practices in encryption and user authentication.

How long does it take to detect a deepfake?

The detection time may vary depending on the file size and type, but typically it takes only a few seconds to process.

Can I upload multiple files at once?

Currently, you can only upload one file at a time for detection. We are working on adding support for multiple file uploads.

Do I need an account to use this service?

Yes, you need to create an account to upload files and view detection results. This helps us ensure data security and provide a personalized experience.

What should I do if I suspect a video is a deepfake?

If you suspect a video is a deepfake, you can upload it to our platform for analysis. If confirmed, please report it to the appropriate authorities or platform administrators.
